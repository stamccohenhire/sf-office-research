# sf-office-research

## Overview
This project focuses on researching co-working spaces near Union Square, San Francisco. The goal is to provide insights on the availability, amenities, and environments of these spaces.

## Features
- List of co-working spaces
- Analysis of pricing and services
- User reviews and ratings

## Tech Stack
- Markdown for documentation

## Usage
Clone this repository and navigate to the project directory. Use the provided markdown files for information.

## Future Plan
Expand the research to include more locations and user testimonials.
